# BlockWorlds

A custom MC Implementation in C

## Quick start

Needs a C11 compiler, OpenGL, and GLFW3

```sh
# macOS
brew install glfw pkg-config

# Debian/Ubuntu
sudo apt install build-essential pkg-config libglfw3-dev libasound2-dev

make        # build → build/blockworlds
make run    # build and launch
```
